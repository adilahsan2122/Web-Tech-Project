# Web Technology Project

This repo contains all the necessary file and information about our **Web Tech Project** course taken on **3/1**. In this Project, we build an system(model) which will predict *potato disease* based on their images.The motivation behind the project is lies on **Data Science** course which we taken on 2/2. From this course , we knew about CNN(convolution Neural Network) and seems interesting to us. We decided to do a project on this.<br>

**Technology Stack**<br>
Frontend - *React JS*,*React Native*<br>
Backend  - *FastApi*,*tf serving*<br>
Dataset  - [Plant Village Dataset](https://www.kaggle.com/datasets/arjuntejaswi/plant-village)<br>
